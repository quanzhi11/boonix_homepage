项目简介

Booxin 启动器是用于简化游戏安装与启动的工具，旨在提供稳定、可配置的启动体验和便捷的版本管理。当前仍在持续开发中，可能存在未修复的问题。欢迎反馈和贡献。

系统要求

  • 最低：Windows 10（64 位）
  • 推荐：Windows 10/11（64 位），4GB 及以上可用内存
  • 如果游戏依赖 Java，请确保安装了 Java 8 或更高版本，并在设置中正确指定 Java 路径。

下载与安装

  1. 在下载页选择所需版本并点击“下载”。
  2. 若下载的是 `.exe` 文件，保存到本地后双击运行开始安装或直接运行便携版。
  3. 如果浏览器点击下载无响应，可尝试：使用不同浏览器、右键另存为，或直接打开网盘链接下载（2.0.0 测试版的网盘：https://pan.quark.cn/s/2cfab72bee8d）。
  4. 遇到 Windows SmartScreen 提示“受保护的计算机已阻止运行应用”，请点击“更多信息”并选择“仍要运行”以继续（仅在确认来源可信时才这样做）。

界面与设置说明

注意：以下说明基于最新版本的 Booxin 启动器，不同版本的界面可能会有所差异。

主界面

主界面是您使用 Booxin 启动器的主要入口，包含以下核心功能区域：

  • 游戏列表：显示已安装的游戏，支持快速启动和管理。
  • 版本选择器：允许您在不同游戏版本之间切换。
  • 启动按钮：一键启动当前选中的游戏版本。
  • 设置入口：进入详细设置界面。
  • 更新检查：自动检查启动器更新。（待实现）

设置界面

设置界面提供了丰富的配置选项，让您可以根据自己的需求自定义启动器：

  • 基本设置：语言选择、主题切换、启动项设置等。（马上实现）
  • 游戏设置：Java 路径配置、内存分配、启动参数调整。
  • 下载设置：下载目录、同时下载数量、代理设置。
  • 高级设置：日志级别、调试模式、开发者选项（待实现）。

版本管理

版本管理功能允许您轻松管理和切换不同的游戏版本：

  • 版本下载：一键下载所需版本。
  • 版本删除：清理不需要的旧版本。

常见问题与解决办法

警告：在尝试以下解决方案前，请确保您已备份重要数据。

启动问题

  • 下载后无法启动：尝试以管理员身份运行、检查防病毒软件是否隔离了文件，或重新下载文件并关闭浏览器扩展后重试。
  • SmartScreen 或杀软报毒：若确定来源可靠，可选择允许运行；如不确定，请先不要运行并联系开发者提供文件哈希以供核验。

游戏运行问题

  • 游戏卡顿或崩溃：检查分配给游戏的内存是否足够，更新显卡驱动，并在设置中尝试调整启动参数。
  • 找不到 Java：请安装合适版本的 Java，并在启动器设置中手动指定 Java 可执行文件路径（例如：C:\Program Files\Java\jre1.8.0_xxx\bin\java.exe）。

下载问题

  • 下载按钮无响应：可能是网络问题或浏览器阻止弹窗，尝试右键“在新标签页中打开链接”或复制链接到新窗口打开。

故障排查与日志收集

若遇到无法解决的问题，请在反馈时尽量附上：

  • 操作系统版本（例如：Windows 10 21H2）
  • 启动器版本（下载页显示的版本号）
  • 复现步骤、错误提示（完整的文字或截图）
  • 如果有日志文件（如程序目录下的 logs 文件夹），请一并附上
